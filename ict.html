Specialized Subjects in the Information and Communications Technology (ICT) Strand

The Information and Communications Technology (ICT) Strand provides students with the necessary skills, competencies, and values in the areas of system designing, technology application, and customer service. With a focus on computer systems, programming, and telecommunications, students will gain the expertise to succeed in the fast-growing tech industry.

List of Subjects

Specialization Description
Animation (NC II) Learn the basics of creating animated content, including 2D and 3D animation, using industry-standard software and techniques.
Broadband Installation (Fixed Wireless Systems) (NC II) Gain skills in installing fixed wireless broadband systems, including the setup, maintenance, and troubleshooting of wireless technology.
Computer Programming (.Net Technology) (NC III) Learn advanced computer programming using .Net technology, covering application development, software design, and debugging techniques.
Computer Programming (Java) (NC III) Learn Java programming, focusing on object-oriented concepts, software development, and the creation of Java-based applications.
Computer Programming (Oracle Database) (NC III) Develop expertise in Oracle Database programming, including database design, query optimization, and data management techniques.
Computer Systems Servicing (NC II) Gain the skills required to service computer systems, including diagnostics, maintenance, and troubleshooting of hardware and software components.
Contact Center Services (NC II) Learn how to provide customer support in contact centers, including communication skills, call handling, and customer service protocols.
Illustration (NC II) Develop skills in digital illustration, including the use of graphic design tools to create high-quality artwork for various media formats.
Medical Transcription (NC II) Learn the basics of medical transcription, focusing on listening and transcribing medical dictations accurately for healthcare professionals.
Technical Drafting (NC II) Study technical drafting principles and techniques, including the use of software tools for creating accurate and detailed engineering drawings.
Telecom OSP and Subscriber Line Installation (Copper Cable/POTS and DSL) (NC II) Learn to install and maintain telecom systems, including copper cable and subscriber line setups for voice and data communication.
Telecom OSP Installation (Fiber Optic Cable) (NC II) Gain knowledge in installing fiber optic cables for telecom services, focusing on installation techniques, splicing, and system maintenance.
